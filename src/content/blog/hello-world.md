## 欢迎来到我的博客 🎉

这是我使用 **Astro** 框架搭建的个人博客的第一篇文章。

### 为什么选择 Astro？

1. **极致性能** — 默认零 JavaScript，页面加载飞快
2. **Markdown 原生支持** — 写文章就像写笔记一样自然
3. **灵活的部署** — 支持 GitHub Pages、Vercel、Netlify 等多种平台
4. **岛屿架构** — 只在需要的地方加载交互组件

### 代码示例

```javascript
// 这是一个简单的 Astro 组件
---
const greeting = "你好世界";
---
<h1>{greeting}</h1>
```
